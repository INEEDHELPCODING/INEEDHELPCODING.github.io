The first step is to set up a GitHub account. GitHub is essentially a website allowing you to save your files online, acting as history control.
It comes with the feature of hosting a website, which is how I hosted this. You should do more research on this.

Then, install an IDE. I used VS Code, which is very versatile. To use VS Code, install the appropriate extensions for JavaScript, Git Bash, HTML, CSS.
You should also install Git Bash and Git, which is what allows you to save things onto GitHub.

Once you do that, you should learn to code in JavaScript and HTML, the key components for a website. CSS is another, but it is used exclusively for decorating a page.
To learn about how JavaScript works, you might want to learn a little about OOP syntax and general coding concepts.

You should then learn to use the Git terminal and learn some file management, and create a repository on GitHub after you made your account.
Link and save all your files to your GitHub repository, and follow the steps to making a GitHub Page.
